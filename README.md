# Hand Betting Game

A web-based Mahjong tile betting game built with React, TypeScript, and Vite.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## What is handwritten vs. AI-assisted

- Handwritten: game rules, state model, UI structure, CSS, and documentation.
- AI-assisted: initial scaffolding and iterative refinement of the project structure, then I validated and tightened the implementation in code.

## Notes

- High scores are stored locally in the browser.
- The game uses Mahjong number, wind, and dragon tiles with persistent tile values for non-number tiles.
